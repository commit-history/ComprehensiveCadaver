# Media

This folder is a placeholder. The actual media files (images, audio) are **not tracked in this repository** — they live in your Anki collection, which already has them once you've installed the original [Comprehensive Cadaver](https://www.reddit.com/r/medicalschoolanki/comments/1b5xt0q/comprehensivecadaver_the_ultimate_cadavericbased/) deck. You do **not** need to populate this folder to import the translations.

## Optional: make this repo standalone

If you'd like this repository to be self-contained (e.g. for backup, sharing, or importing without the original deck), copy the deck's media files into this folder. Anki's media folder lives at:

- **macOS:** `~/Library/Application Support/Anki2/<profile>/collection.media/`
- **Linux:** `~/.local/share/Anki2/<profile>/collection.media/`
- **Windows:** `%APPDATA%\Anki2\<profile>\collection.media\`

The files themselves stay git-ignored.
